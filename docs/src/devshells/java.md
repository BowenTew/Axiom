# Java

Java 开发环境模板，提供三个 JDK 版本。

## Java 8

```sh
nix flake init -t github:BeauvnTu/Axiom#java8
```

| 包 | 说明 |
|----|------|
| `jdk8` | Java 8 JDK |
| `maven` | Maven 构建工具 |
| `gradle` | Gradle 构建工具 |
